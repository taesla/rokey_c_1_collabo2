다수 팀 동시 작업
실습실에서 5개 팀이 동시에 작업할 때,
ID로 네트워크를 논리적으로 분리합니다.
Team 1: ID=60
Team 2: ID=61
Team 3: ID=62
...
팀 내 협업 (N대 연결)
한 팀 안에서 노트북 여러 대를 연결하여
제어, 비전, 모니터링을 분담합니다.
Laptop A: ID=60
Laptop B: ID=60
Robot: ID=60
# 모두 동일!
개인 개발 / 시뮬레이션
혼자 작업할 때도 일관성을 위해
팀 ID를 유지하거나 기본값을 사용합니다.
# 집에서 작업 시
export ROS_DOMAIN_ID=60
# 또는
unset ROS_DOMAIN_ID
