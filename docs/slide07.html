ROS2 분산 통신 기술

DDS Discovery
1. 토픽 공유 (Topic)
노트북 A가 발행한 메시지를
나머지 노트북이 실시간 구독합니다.
설정 없이 자동 발견(Discovery)됩니다.
# Laptop A
pub('/hello', "Hi")

# Laptop B, C, D
sub('/hello') -> "Hi"
2. 서비스 호출 (Service)
노트북 A에 있는 로봇 제어 함수를
노트북 B에서 원격으로 호출하여
실행 결과를 받아볼 수 있습니다.
# Laptop A (Server)
Service('move_robot')

# Laptop B (Client)
Call('move_robot')
3. 분산 처리 (Distributed)
무거운 연산(영상처리, SLAM)을
여러 노트북에 나누어 처리하여
전체 시스템 성능을 높입니다.
Laptop 1: Robot Driver
Laptop 2: Camera Node
Laptop 3: Path Planning
