Final Checklist
Best Practices
  • 조 내의 모든 장비는 동일 ID (60) 사용
  • 역할 분담: 제어(Main)와 모니터링(Sub) 분리
  • 데이터 수집 시 부하 분산 (Load Balancing)
  • 시작 전 'ros2 topic list'로 연결 확인
Critical Warnings
  • 다른 WiFi 접속 금지: 4대 모두 동일망 필수
  • 방화벽(Firewall): Ubuntu ufw가 ROS2 차단 가능
  • 대역폭: 카메라 4대 동시 전송 시 지연 발생
