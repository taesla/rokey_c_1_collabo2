트러블슈팅 (Troubleshooting)

Q & A
증상: 다른 팀의 토픽이 보임
1. 현재 ID 확인 (0이면 모두 보임)
echo $ROS_DOMAIN_ID
2. ID 재설정 및 노드 재시작
export ROS_DOMAIN_ID=60; pkill -f ros2
증상: 팀원끼리 통신 안 됨
1. 양쪽 노트북 ID 일치 확인
2. 같은 WiFi 접속 확인 (Ping 테스트)
3. 방화벽 해제
sudo ufw disable
증상: 터미널 켜면 설정 사라짐
1. .bashrc 파일에 저장 안 됨
2. 영구 설정 명령어 실행
echo 'export ROS_DOMAIN_ID=60' >> ~/.bashrc
증상: ID 바꿨는데 이전 토픽이 보임
1. 이전 프로세스가 살아있는 경우
2. 강제 종료 후 재시작
pkill -f ros2; ros2 daemon stop; ros2 daemon start
