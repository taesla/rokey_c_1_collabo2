왜 개별 설정이 필요한가?

Problem & Solution
잘못된 방법 (충돌)
export ROS_DOMAIN_ID=60 # Team 1
export ROS_DOMAIN_ID=60 # Team 2 (Conflict!)
export ROS_DOMAIN_ID=60 # Team 3 (Conflict!)
다른 팀 명령이 내 로봇 제어
토픽 데이터 혼선 및 오염
디버깅 불가능 수준의 혼란
올바른 방법 (격리)
export ROS_DOMAIN_ID=60 # Team 1
export ROS_DOMAIN_ID=61 # Team 2 (Isolated)
export ROS_DOMAIN_ID=62 # Team 3 (Isolated)
팀별 독립적인 작동 보장
타 팀 데이터 보이지 않음
안전하고 쾌적한 개발 환경
