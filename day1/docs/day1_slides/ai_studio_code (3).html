Topic 2: RG2 Gripper & Modbus 통신

이론: Modbus 통신

산업용 장비 제어에 널리 쓰이는 표준 시리얼 통신 프로토콜입니다.

Master-Slave 구조

Master (PC)

Request →

← Response

Slave (Gripper)

Master가 항상 먼저 요청하고, Slave는 그에 응답합니다.

실습: 그리퍼 제어

네트워크 설정 (PC에 그리퍼용 IP 추가):

sudo nmcli connection modify "Wired connection 2" \
ipv4.addresses 192.168.1.100/24 ipv4.method manual

제어 코드 (pymodbus 2.5.3):

from onrobot import RG
gripper = RG('192.168.1.1')
gripper.open_gripper()
gripper.move(500) # 50mm
