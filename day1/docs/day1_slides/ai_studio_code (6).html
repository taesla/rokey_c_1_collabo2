ROS_DOMAIN_ID 설정

ROS2 DDS 통신에서 네트워크를 격리하여 다른 팀과의 충돌을 방지합니다.

# ~/.bashrc에 추가
echo 'export ROS_DOMAIN_ID=60' >> ~/.bashrc
source ~/.bashrc

주요 트러블슈팅

  • 카메라 접근 오류: ROS 노드 종료 후 OpenCV 스크립트 실행
  • 캘리브레이션 `nan` 값: 데이터 재수집 (조명, 각도 조절)
  • 그리퍼 연결 실패: `ip addr add`로 서브넷 추가
  • pymodbus 버전 오류: 2.5.3 버전으로 다운그레이드
