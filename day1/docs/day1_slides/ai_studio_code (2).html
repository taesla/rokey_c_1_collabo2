Topic 1: RealSense 카메라

이론: D435 구성 요소

Color Camera
RGB 컬러 이미지 캡처, 3D Reconstruction에 사용
Depth Sensor
두 개의 IR 카메라로 시차(Disparity)를 이용해 깊이 계산
IR Projector
적외선 패턴을 투사하여 Texture 없는 표면의 깊이 측정 보조

실습: ROS2 연동

RealSense 런치 실행:

ros2 launch realsense2_camera rs_launch.py align_depth.enable:=true

주요 토픽 확인:

/camera/color/image_raw

/camera/depth/image_rect_raw

/camera/aligned_depth_to_color/image_raw
