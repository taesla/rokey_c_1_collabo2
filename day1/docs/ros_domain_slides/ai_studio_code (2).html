설정 방법 (Setup Guide)

Configuration
임시 (Temporary)
1. 현재 터미널만
# 팀 번호에 맞게 설정 (예: 60)
$ export ROS_DOMAIN_ID=60

# 설정 확인
$ echo $ROS_DOMAIN_ID
> 60
단점: 터미널을 닫으면 설정이 사라집니다.
영구 (Permanent) - 권장
2. .bashrc 파일 수정
# .bashrc 파일 열기
$ nano ~/.bashrc

# 맨 아래에 추가
export ROS_DOMAIN_ID=60

# 저장 후 적용
$ source ~/.bashrc
장점: 새 터미널을 열어도 설정이 유지됩니다.
